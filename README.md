## 📖 About
This repository contains the source code for the official Cloud Bucket landing page. Cloud Bucket is a unified cloud storage manager that allows Windows users to connect, view, and manage their Google Drive and Microsoft OneDrive accounts from a single, beautiful desktop dashboard.

This website serves two primary purposes:
1. **Landing Page:** A professional destination for users to learn about the app and download the Windows client.
2. **App Verification:** Hosting the required Privacy Policy (`privacy.html`) and Terms of Service (`terms.html`) necessary for Google Cloud OAuth and Microsoft Entra App Verification compliance.

## 🚀 Features
* **Modern Design:** Built with semantic HTML5 and vanilla CSS for lightning-fast load times.
* **Responsive:** Fully responsive layout that looks great on desktop and mobile devices.
* **SEO Optimized:** Structured for search engines with meta tags and dynamic XML sitemaps.
* **Edge Hosting:** Deployed globally via Cloudflare Pages for maximum uptime and minimal latency.
